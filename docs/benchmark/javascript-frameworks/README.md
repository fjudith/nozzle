# Size of Javascript Framework

## Synopsis

Leveraging Javascript Frameworks for application Frontend introduce a notable overhead in term of performance and thus compute comsumtion.
This benchmark aims to evaluate the passive impact of popular frameworks

* Angular
* React
* [Vue.JS](vuejs.org)
